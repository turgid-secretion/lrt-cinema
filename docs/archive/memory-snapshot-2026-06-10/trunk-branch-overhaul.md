Pipeline overhaul to the **trunk/branch model**. Branch `feat/trunk-branch-overhaul` (off feat/highlight-recovery-tier1). Plan: `docs/research/pipeline-overhaul-plan.md` (binding). Motivating analysis: `docs/research/pipeline-worldclass-gap-and-plan.md` + `pipeline-order-audit.md`.

**THE MODEL (the conceptual spine — settled with the owner):** the pipeline is TWO products with OPPOSITE definitions of "correct" sharing one machine. Realize as a **scene-linear data-preserving TRUNK** (demosaic→recon→WB→camera-colour→HSM→ExposureRamp = tap-7, already exists as cinema-linear-master) that BRANCHES: (a) **faithful branch** applies the Adobe look (LookTable→ProfileToneCurve→develop ops, Adobe order) → sRGB; lossy BY MANDATE (matches a lossy target — correct, not a defect). (b) **master branch** taps the clean trunk. KEY INSIGHT from the owner (don't lose it): this is a PER-FRAME moving-image pipeline; per-frame intent (deflicker, Holy-Grail exposure ramp, keyframed ops) has **NO transport across the NLE handoff** (DECISIONS §4 — keyframes die in Resolve import), so it MUST bake into pixels — that's the raison d'être of LRT/lrt-cinema. So the real axis is **temporally-VARYING (must bake) vs temporally-CONSTANT (defer-able)**, NOT "clean vs developed". The "single working space / look-at-the-end" ideal SURVIVES for the master but BREAKS for the faithful branch (its look is mid-chain, non-separable, lossy by mandate); the one irreducible seam = Adobe's LookTable is authored on [0,1] and can't carry overrange, so you can't be both data-preserving AND Adobe-faithful through it.

**OWNER DECISIONS (binding):** (Q1) per-frame always bakes; `--master-look {bake,defer}` (default **defer**=clean master) gates the static Stage-12 look on the perceptual master. (Q2) **DCB demosaic now** (flagged); clean-room RCD numba port = follow-up (AMaZE=GPL3, LMMSE=GPL2, RCD=absent in installed libraw 0.22.1; DCB/AHD/DHT are LGPL-core, ~5dB>bilinear). (Q3) **flag-gated; defaults byte-exact; OWNER validates** the real gates before flipping defaults.
