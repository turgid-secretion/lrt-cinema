# Local render fixtures — locations & regeneration

Owner-approved: 2026-06-10

The render/ΔE regression tests need large external fixtures (DNGs, Adobe
reference TIFFs, the `dng_validate` binary). They are NOT committed. The
canonical local home is **`~/lrt-cinema-fixtures/`** (override with the
`LRT_CINEMA_FIXTURES` env var). The historical `/tmp/dng_out` location is
still honoured as a fallback — but **macOS wipes /tmp**, which is how the
gym end-to-end gate silently went dormant. Resolution logic:
[tests/fixture_paths.py](tests/fixture_paths.py).
